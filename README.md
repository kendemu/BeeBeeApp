# BeeBeeApp  
App for Bee Bee Bungalows.  
## Usage  
1. Setup restaurant menus.  
1.1. Make [menucategories].txt at menus/ and enter a list of menus.  
1.2. Run menutodb.py  
```
./menutodb.py
```
2. Start Server.  
2.1. Build the server code.  
```
npm run build
```
2.2. Build the Web Page.  
```
npm run build-pages
```
2.3. Run server.  
```
npm run start
```
